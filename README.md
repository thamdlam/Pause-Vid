# Pause Vid: The Ultimate Short & Long-Form Video Platform

Pause Vid is an open-source, privacy-first alternative to TikTok, Likee, and YouTube. It combines real-time short-form video algorithms with robust long-form video hosting.

## 🚀 Core Features
* **Dual-Feed System:** Seamlessly switch between micro-content and deep-dive videos.
* **Creator Studio:** Built-in filters, audio mixing, and AI-assisted video editing tools.
* **Live Streaming:** Low-latency interactive streaming with virtual gifting capabilities.
* **Decentralised Storage:** Optional Web3 integration using IPFS for video hosting.

## 🛠️ Tech Stack
* **Frontend:** Flutter / React Native (Cross-platform iOS & Android)
* **Backend:** Node.js (Microservices) / Go (Video streaming engine)
* **Database:** PostgreSQL (User data) / Redis (Caching & Feed queues)
* **Video Processing:** FFmpeg / AWS Elemental MediaConvert

## 🏁 Getting Started
1. Clone the repository: `git clone https://pausevid.com`
2. Install dependencies: `npm install`
3. Set up your `.env` file using `.env.example`.
4. Run the development server: `npm run dev`
